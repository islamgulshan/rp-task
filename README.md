## Description

This is a simple CRUD application for managing cars. It is built using NestJS, React, and MySQL.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev
```

## Api Documentation

```bash
# Categories
$ http://localhost:3000/api/categories

# Cars
$ http://localhost:3000/api/cars
```
